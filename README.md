# Hardhat project to create ERC721 contract (nft)

This project demonstrates a basic Hardhat use case. It comes with a ERC721 contract.
Try running some of the following tasks:

```shell
npx hardhat help
npx hardhat node
npx hardhat run scripts/deploy.js
```
